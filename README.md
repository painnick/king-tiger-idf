# RC 탱크 프로젝트

Bluepad32를 이용한 게임패드 조작 RC 탱크입니다. ESP32-IDF(v5.4.4) 기반으로 개발되었습니다.

## 주요 기능

- **게임패드 연결**: Bluepad32를 통한 블루투스 게임패드 연결 및 상태 LED/진동 피드백
- **트랙 제어**: 좌측 스틱 Y축(좌측 트랙), 우측 스틱 Y축(우측 트랙) 독립 제어 (DRV8833 #1)
- **터렛 제어**: D-PAD 좌우로 터렛 회전 (DRV8833 #2)
- **포 마운트 제어**: D-PAD 상하로 포신 상하 이동 (DRV8833 #2)
- **포신 발사**: B 버튼 클릭 시 포신 반동(서보 90->180->90), LED 점등, 발사 효과음 및 게임패드 진동 발생 (트랙 후진 반동 300ms 동기화)
- **기관총 발사**: A 버튼 클릭 시 기관총 LED 깜빡임(500ms), 효과음 및 게임패드 진동 발생 (반응성 개선을 위해 지연 시간 300ms로 단축)
- **헤드라이트**: Y 버튼으로 헤드라이트 ON/OFF 토글
- **볼륨 조절**: L1 버튼으로 감소, R1 버튼으로 증가 (NVS 저장, 10-30 범위)
- **설정 초기화**: Select + Start 버튼을 3초간 누르면 볼륨 설정 초기화 및 재시작

## 하드웨어 구성

### ESP32 핀 맵

| ESP32 핀 | 기능 | 설명 |
| :--- | :--- | :--- |
| **GPIO 4** | 후미등 LED | Backlight |
| **GPIO 5** | 헤드라이트 LED | Headlight (Y 버튼 토글) |
| **GPIO 16** | 기관총 LED | MG LED (A 버튼 클릭 시 깜빡임) |
| **GPIO 17** | 포신 LED | Gun LED (B 버튼 클릭 시 점등) |
| **GPIO 18** | 포 마운트 IN2 | DRV8833 #2 - 포 마운트 상하 |
| **GPIO 19** | 포 마운트 IN1 | DRV8833 #2 - 포 마운트 상하 |
| **GPIO 21** | 터렛 IN2 | DRV8833 #2 - 터렛 회전 |
| **GPIO 22** | 터렛 IN1 | DRV8833 #2 - 터렛 회전 |
| **GPIO 23** | 사운드 TX | JQ6900 UART TX |
| **GPIO 25** | 우측 트랙 IN2 | DRV8833 #1 - 우측 트랙 |
| **GPIO 26** | 좌측 트랙 IN2 | DRV8833 #1 - 좌측 트랙 |
| **GPIO 27** | 좌측 트랙 IN1 | DRV8833 #1 - 좌측 트랙 |
| **GPIO 32** | 포신 서보 | 포신 반동 (Recoil) 제어 |
| **GPIO 33** | 우측 트랙 IN1 | DRV8833 #1 - 우측 트랙 |

### 모터 드라이버 (DRV8833) 연결

#### 1번 드라이버 (트랙)
- **AIN (27, 26)**: 좌측 트랙 모터
- **BIN (33, 25)**: 우측 트랙 모터

#### 2번 드라이버 (액세서리)
- **AIN (22, 21)**: 터렛 회전 모터
- **BIN (19, 18)**: 포 마운트 상하 모터

### 서보 모터
- **GPIO 32**: 포신 반동 서보 (B 버튼 시 90도 -> 180도 -> 90도 동작)

### 사운드 모듈 (JQ6900/JQ6500)
- **TX (GPIO 23)**: 모듈의 RX 핀에 연결하여 UART 제어 (9600bps)

## 게임패드 조작법

| 조작 | 기능 | 상세 동작 |
| :--- | :--- | :--- |
| **좌측 스틱 Y** | 좌측 트랙 | 위(전진), 아래(후진) |
| **우측 스틱 Y** | 우측 트랙 | 위(전진), 아래(후진) |
| **D-PAD 좌/우** | 터렛 회전 | 좌우 회전 (DRV8833) |
| **D-PAD 상/하** | 포 마운트 | 상하 이동 (DRV8833) |
| **A 버튼** | 기관총 발사 | 효과음 재생 + MG LED 깜빡임 |
| **B 버튼** | 주포 발사 | 효과음 재생 + Gun LED 점등 + 서보 반동 |
| **Y 버튼** | 헤드라이트 | ON / OFF 토글 |
| **L1 / R1** | 볼륨 조절 | 볼륨 감소 / 증가 (NVS 자동 저장) |
| **Select + Start** | 시스템 초기화 | 3초간 입력 시 설정 초기화 및 재부팅 |

## 소프트웨어 구조

### 주요 컴포넌트 (`components/rctank/`)
- `rctank_motor`: MCPWM을 이용한 트랙 제어 및 GPIO를 이용한 액세서리(터렛/포 마운트) 제어
- `rctank_servo`: LEDC를 이용한 포신 서보 제어
- `rctank_led`: 각종 LED 상태 제어
- `rctank_dfplayer`: JQ6900/JQ6500 UART 통신 및 효과음 재생
- `rctank_storage`: NVS를 이용한 볼륨 및 설정값 저장

### 플랫폼 로직 (`main/my_flatform.c`)
- Bluepad32 이벤트 핸들링 및 게임패드 입력 맵핑
- 발사 시퀀스 지연 보정 (Sound 재생 시점과 하드웨어 동작 동기화)

## 개발 환경 설정

1. **ESP-IDF v5.4.4** 설치
2. 프로젝트 루트의 `env.bat`를 실행하여 환경 변수 설정 (Windows)
3. `idf.py build` 명령으로 컴파일
4. `idf.py flash monitor` 명령으로 업로드 및 로그 확인
