# RC 탱크 프로젝트

Bluepad32를 이용한 게임패드 조작 RC 탱크입니다.

## 기능

- **게임패드 연결**: Bluepad32를 통한 블루투스 게임패드 연결
- **트랙 제어**: 좌측 스틱 Y축으로 좌측 트랙, 우측 스틱 Y축으로 우측 트랙 독립 제어
- **터렛 제어**: D-PAD 좌우로 터렛 회전
- **포 마운트 제어**: D-PAD 상하로 포 마운트 DC 모터 상·하 구동 (DRV8833)
- **포신 발사**: B 버튼으로 포신 당기기 (LED 켜짐 + 효과음 2, 200ms)
- **기관총 발사**: A 버튼으로 기관총 발사 (LED 동작 없음 + 효과음 3, 1초간)
- **헤드라이트**: Y 버튼으로 헤드라이트 ON/OFF (100ms 간격)
- **볼륨 조절**: L1 버튼으로 볼륨 감소, R1 버튼으로 볼륨 증가 (EEPROM 저장, 100ms 간격)
- **EEPROM 초기화**: Select + Start 버튼 3초간 누르면 모든 설정 초기화 및 재시작
- **효과음**: JQ6500 모듈을 통한 SD카드 효과음 재생

## 하드웨어 연결

### ESP32 핀 연결

| ESP32 핀 | 연결 대상 | 설명 |
|---------|-----------|------|
| GPIO 12 | (미사용) | **유의**: 부팅 시 플래시 전압 이슈 가능 |
| GPIO 13 | (미사용) | 일반 GPIO |
| GPIO 14 | (미사용) | **유의**: 부팅 시 신호 발생 장치 비권장 |
| GPIO 15 | (미사용) | **유의**: 부팅 시 로그 및 신호 발생 가능 |
| GPIO 16 | 기관총 LED | 기관총 발사 시 점등 |
| GPIO 17 | 헤드라이트 LED | 헤드라이트 제어 |
| GPIO 23 | 포신 서보 모터 | 포신 당기기 제어 (B 버튼) |
| GPIO 5  | 포신 LED | 포신 발사 시 점등 (부팅 스트랩 유의) |
| GPIO 32 | 좌측 트랙 IN1 | DRV8833 #1 좌측 트랙 (AIN1) |
| GPIO 25 | 좌측 트랙 IN2 | DRV8833 #1 좌측 트랙 (AIN2) |
| GPIO 26 | 우측 트랙 IN1 | DRV8833 #1 우측 트랙 (BIN1) |
| GPIO 27 | 우측 트랙 IN2 | DRV8833 #1 우측 트랙 (BIN2) |
| GPIO 19 | 포 마운트 IN1 | DRV8833 #2 포 마운트 DC (BIN1) |
| GPIO 18 | 포 마운트 IN2 | DRV8833 #2 포 마운트 DC (BIN2) |
| GPIO 22 | 터렛 모터 IN1 | DRV8833 #2 터렛 (AIN1) |
| GPIO 21 | 터렛 모터 IN2 | DRV8833 #2 터렛 (AIN2) |
| GPIO 33 | JQ6500 RX | JQ6500 UART 통신 (ESP32 TX) |

(본 프로젝트에서는 온칩 JTAG 디버깅을 사용하지 않으므로 GPIO 12~15번은 일반 용도로 활용 가능합니다.)

### GPIO 맵 검토 (ESP32, WROOM/WROVER 일반 구성 기준)

| 구분 | GPIO | 용도 | 검토 메모 |
|------|------|------|-----------|
| **GPIO 확장** | **12, 13, 14, 15** | **(미사용)** | JTAG용 핀으로, 사용 시 부팅 스트래핑(12, 15) 조건 확인 필수 |
| **Cluster A (좌측)** | **25, 26, 27, 32** | **좌·우 트랙** | DRV8833 #1 제어용으로 묶음 배치 |
| **Cluster B (우측)** | **22, 21, 19, 18** | **터렛, 포 마운트** | DRV8833 #2 제어용으로 묶음 배치 |
| 서보 | 23 | 포신 | 우측 최하단 배치 |
| LED | 5, 17, 16 | 포신, 헤드라이트, MG | 출력 전용 (5번은 부팅 시 안전 고려 포신 LED) |
| UART | 33 | JQ6500 | 단방향 제어(TX Only) |

**사용 금지(일반 모듈):** GPIO **6~11**은 내장 SPI 플래시 점유—GPIO 할당에서 제외한다.

**스트랩핑:** GPIO **5**는 부팅 시 상태 샘플링 이슈가 있으므로 모터가 아닌 **LED** 제어용으로 사용하여 오동작을 방지한다.

**입력 전용:** GPIO **34~39**는 출력 불가—모터 드라이버 IN선·LED 구동에 쓰지 않는다.

**배선 정합성:** 위 표의 “포 마운트 IN1/IN2”는 아래 DRV8833 절의 **BIN1/BIN2**(GPIO 19·18)와 동일 배선을 가리킨다. AIN/BIN과 모터 극성은 기계 방향에 맞게 교차 검증한다.

**MCPWM:** DC 모터 다채널을 MCPWM으로 돌릴 때는 타이머·연산자·핀 매핑을 한 번에 짜는 것이 좋다(구현 단계 과제).

### DRV8833 모터 드라이버 연결

#### 좌측 트랙 (DRV8833 #1)
- AIN1 → GPIO 32
- AIN2 → GPIO 25
- AOUT1 → 좌측 트랙 모터 1
- AOUT2 → 좌측 트랙 모터 2
- VM → 12V 전원
- GND → 공통 접지
- VCC → 3.3V (논리 전원)

#### 우측 트랙 (DRV8833 #1)
- BIN1 → GPIO 26
- BIN2 → GPIO 27
- BOUT1 → 우측 트랙 모터 1
- BOUT2 → 우측 트랙 모터 2

#### 터렛 및 포 마운트 (DRV8833 #2)
- **터렛 (A채널)**
  - AIN1 → GPIO 22
  - AIN2 → GPIO 21
  - AOUT1 → 터렛 모터 +
  - AOUT2 → 터렛 모터 −
- **포 마운트 (B채널, DC 모터)**
  - BIN1 → GPIO 19
  - BIN2 → GPIO 18
  - BOUT1 → 포 마운트 DC 모터 +
  - BOUT2 → 포 마운트 DC 모터 −
- VM → 12V 전원
- GND → 공통 접지
- VCC → 3.3V (논리 전원)

### 서보 모터 연결

#### 포신 서보 모터 (당기기)
- **신호선**: GPIO 23
- **전원**: 5V
- **접지**: 공통 접지
- **기능**: B 버튼으로 포신 당기기

### JQ6500 모듈 연결

- 모듈 **RX** → GPIO 33 (ESP32 UART TX)
- 모듈 **TX** → (미연결)
- **VCC** → 5V (보드 매뉴얼 기준, 일부 보드는 3.3V만 허용할 수 있음)
- **GND** → 공통 접지
- **스피커 출력**(SP+, SP− 등 모듈 표기) → 스피커

**통신:** 펌웨어는 JQ6500 모듈 데이터시트의 UART 명령 형식에 맞게 구현한다.

## SD카드 설정

SD카드에 다음 파일들을 저장하세요:

```
sdcard/
├── 0001.mp3  (효과음 1 - 대기 시 반복 재생)
├── 0002.mp3  (효과음 2 - 포신 발사 시 재생)
├── 0003.mp3  (효과음 3 - 기관총 발사 시 재생)
└── 0004.mp3  (효과음 4 - 게임 패드 연결 시 재생)
```

## 게임패드 조작법

### 기본 조작
- **좌측 스틱 Y축**: 좌측 트랙 전후진 제어
- **우측 스틱 Y축**: 우측 트랙 전후진 제어
- **D-PAD 좌우**: 터렛 좌우 회전
- **D-PAD 상하**: 포 마운트 DC 모터 상·하 구동 (듀티/속도는 펌웨어에서 설정; 기계적 한계는 리미트·시간 제한 등으로 관리)

### 버튼 조작
- **B 버튼**: 포신 당기기 (LED 켜짐 + 효과음 2 재생, 200ms)
- **A 버튼**: 기관총 발사 (LED 동작 없음 + 효과음 3 재생, 1초간)
- **Y 버튼**: 헤드라이트 ON/OFF 토글



### 볼륨 조절
- **L1 버튼**: 볼륨 감소 (10-30 범위, 100ms 간격)
- **R1 버튼**: 볼륨 증가 (10-30 범위, 100ms 간격)

*볼륨 설정은 버튼을 뗐을 때 EEPROM에 자동 저장되어 다음 시작 시에도 유지됩니다.*

### 고급 설정

- **Select + Start 버튼 3초간 누르기**: 모든 EEPROM 설정 초기화 및 ESP32 재시작

### LED 동작
- **포신 발사 시**: 포신 LED 지속 점등 (200ms 동안)
- **기관총 발사 시**: 포신 LED 동작 없음
- **헤드라이트**: Y 버튼으로 ON/OFF 토글

### 게임패드 피드백
- **포신 발사 시**: 게임패드 진동 (400ms)
- **기관총 발사 시**: 게임패드 진동 (300ms)

- **EEPROM 초기화 시**: 게임패드 진동 (800ms)
- **볼륨 변경 시**: 시리얼 로그로 확인
- **포신 발사 시간**: 200ms로 단축 (기존 500ms에서 개선)

## 컴파일 및 업로드

1. PlatformIO IDE에서 프로젝트를 열기
2. 필요한 라이브러리들이 자동으로 설치됩니다
3. ESP32 보드를 연결하고 업로드
4. 시리얼 모니터로 초기화 과정 확인

## 초기화 과정

1. ESP32 부팅
2. EEPROM에서 볼륨 설정 로드
3. 구동부 초기화 (포 마운트 DC: DRV8833 출력 정지, 포신 서보: 90도)
4. JQ6500 초기화 및 효과음 1 재생 시작
5. Bluepad32 초기화
6. 게임패드 연결 대기

## 게임패드 연결

1. ESP32 부팅 후 블루투스 페어링 모드 진입
2. 게임패드에서 블루투스 설정으로 ESP32 연결
3. 연결 성공 시 효과음 4 재생 및 효과음 1 재생 중단
4. 게임패드 조작 가능

## 문제 해결

### 게임패드가 연결되지 않는 경우
- ESP32 재부팅 후 다시 시도
- 게임패드 페어링 정보 초기화 후 재연결

### 모터가 작동하지 않는 경우
- 전원 공급 확인 (12V)
- DRV8833 연결 확인 (좌측 **25·26**, 우측 **27·32**, 터렛 **22·21**, 포 마운트 **19·18**)
- PWM(MCPWM 등) 채널·핀 매핑 확인
- GPIO **5**는 LED 전용으로 변경됨(부팅 시 모터 방지용)

### 효과음이 나오지 않는 경우
- SD카드 파일명 확인 (0001.mp3, 0002.mp3 등)
- JQ6500 연결 확인 (ESP32 TX 핀: GPIO 33)
- 볼륨 설정 확인 (코드에서 20으로 설정)

### 서보 모터가 작동하지 않는 경우 (포신)
- 포신 서보 전원 공급 확인 (5V)
- 신호선 연결 확인 (GPIO 23)
- ESP32Servo 라이브러리 설치 확인
